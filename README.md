<h3>To install & build ui</h3>
  <ul>
		<li>cd ui/</li>
  	<li>npm install</li>
  	<li>(grunt build)</li>
</ul>

"npm install" downloads required node modules and other dependencies to local cache. 

"grunt buld" builds the app (minification etc.)

<h3>Running the rest API</h3>

bundle install
bundle exec ruby rest.rb
